Subject/ Topic: Image Based Rubick’s Cube Solver (Using Pattern Recognition & Game Play) 

 

Description: Using this project/ algorithm that we’re going to create, we intend to solve a rubick’s cube by pattern recognition and game play and then scale this algorithm to solve higher scale problems (Ex: solving 3x3, 4x4, 5x5, 6x6, etc).  

Pattern recognition 

We use a camera/ 6 images showing different sides of the cube. 

We then use this to identify the cube state before we solve it. 

 

Game Play (Search) 

We solve the cube using the IDA* search algorithm (Iterative deepening A* search).
